# Number

山奈提供了一些数学函数，这些函数可以用于数值计算。

## `formatNumber`

### 介绍

格式化数字，保留两位小数。

### 导入

```js
import { formatNumber } from 'galanga'
```

### 使用

输入的参数如下：

- `number`[number]: 数字

::: code-group

```js [按需引入]
number = formatNumber(number)
```

```js [全局引入]
number = galanga.formatNumber(number)
```

:::
